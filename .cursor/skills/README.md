# Skills en morning-brain-starter

Los skills en esta carpeta indican al asistente **cuándo** y **cómo** usar la rutina matinal y el boarding.

- **good-morning:** resumen de reuniones (Calendar), tareas (Asana opcional), emails (Gmail opcional), bitácoras y alertas opcionales. Comando: `python run_morning.py`.
- **morning-anomalies:** detección GA4/GSC e informes HTML. Ver `docs/ANOMALIES.md`.
- **generate-anomaly-demo-data:** regenerar CSV y controles del demo Tycho.
